## What Was Done

### Built from scratch in one session (vibe-coded)

1. **Flask server** (`server/listen.py`, ~230 lines)
   - Single-file architecture — serves both API and embedded web client
   - Whisper `base` model integration with lazy loading at startup
   - Audio conversion pipeline: receives WEBM from browser → ffmpeg converts to 16kHz mono WAV → feeds numpy array to Whisper
   - Auto-paste function using `pbcopy` + `osascript` keystroke (Cmd+V via System Events AppleScript)
   - Self-signed SSL cert generation with `cryptography` library, auto-created at `~/.remyk-ssl/`
   - Graceful HTTP fallback when `cryptography` isn't installed
   - Routes: `/` (serves embedded client), `/transcribe` (POST audio → returns text + paste status), `/status` (server health JSON)

2. **Embedded web client** (`client/index.html`, 37 lines)
   - SVG microphone button with pulse animation when recording
   - Status display showing recording/transcribing state
   - Result text area with empty-state styling
   - "Copy & close" button for fallback paste
   - Hint text reminding users they can manually Cmd+V if auto-paste misses
   - Focus tip banner for remote desktop situations (Remote Desktop, SSH forwarding, etc.)

3. **Project structure created**
   - `server/` — Flask server code
   - `client/` — Embedded web client (served by server)
   - `docs/` — Documentation placeholder
   - `server-win/` — Windows variant placeholder
